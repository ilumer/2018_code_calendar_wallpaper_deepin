# 2018_code_calendar_wallper_deepin

直接修改的[RayZhao1998][1]的壁纸切换方式 因为分辨率的问题合适的大小需要自己手动调所以这个项目并不是很实用。

```
$ sudo apt-get install libmagickwand-dev
```
